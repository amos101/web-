#概要
ユーザー登録ができて投稿機能のあるSNSアプリです。気に入った投稿には『いいね！』することもできます。
初めてのポートフォリオサイトになります。

#制作背景
ポートフォリオを通して、フロントエンドからバックエンド、インフラについて知見を増やしたく思い作成しました。
アプリケーションをHeroku上のサーバーにデプロイしました。

#機能一覧

*投稿機能
